# KrimBoard-Simple-soundboard
_Simple soundboard like soundux, voicemod, made by DeepSeek._ Довольно простенькая замена таким сервисам как Soundux и VoiceMod, и подобным сервисам. Написан ИИ DeepSeek по моим промптам. Создан по нужде обстоятельств..

Имеет функции загрузки файлов поодиночно, и загрузку целой папки сразу. Может использовать 2 аудиовыхода. Для надёжного сохранения назначенных клавишь используется JSON файл. Имеет возможность проверки аудиовыхода прямо в приложении (короткий звуковой сигнал). 
Также есть 3 режима сортировки файлов: по алфавиту, по дате создания, и ручное изменение (последнее не тестировал).
